# 公開前チェックリスト

## 動作確認

- `node --check app/script.js` が通る。
- `node --check app/romaji-engine.js` が通る。
- `npm run check:public-info` が通る。
- 公開URLでタイトルから `app/` に移動できる。
- 公開URLでゲームが開く。
- 開始ボタンでゲームが始まる。
- タイトルが `モジリアクト` になっている。
- 最初の一言が文字リアクションのコンセプトを伝えている。
- `app/assets/ogp.svg` が表示できる。
- 最初の画面で複数の文字反応プレビューが見える。
- 通常ルートと別ルートの両方で正解になる。
- `しゃ` を含む単語で `sha` と `sya` の両方が通る。
- 小さい `っ` を含む単語で子音重ねと `xtu/ltu` 系が通る。
- 単語の雰囲気に合わせて文字演出が変わる。
- `薄い`, `爆速`, `秘密`, `ごちゃごちゃ`, `雨`, `熱い` 系の反応差が見て分かる。
- プレイ中の背景色や流れが、現在のお題のムードに合わせて変わる。
- 速度、雨、炎、発光、混乱などのムードで、カード内の専用演出が変わる。
- お題が100語あり、すべての読みで入力候補を生成できる。
- お題はエフェクト化しやすい言葉だけになっている。
- ミス入力でミス扱いになり、現在位置が戻る。
- 60秒後に結果画面が出る。
- 結果コピー用の投稿文が表示される。
- 再挑戦で状態が初期化される。

## GitHub Pages

- Pagesの公開元はリポジトリルートを想定する。
- ルートの `index.html` から `app/` に遷移する。
- 公開URLは `https://omicreate.github.io/mojireact-typing-game/`。
- GitHubリポジトリは `https://github.com/omicreate/mojireact-typing-game`。
- 外部API、外部CDN、ビルド工程は不要。
- `.nojekyll` を置き、GitHub Pages側の余計な処理を避ける。
- OGP画像は `app/assets/ogp.svg` を参照する。
