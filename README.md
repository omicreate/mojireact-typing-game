# モジリアクト

ブラウザだけで遊べる、静的なローマ字タイピングゲームです。

コンセプトは「打つたび、ことばの意味が文字に出る」。入力方式対応はゲームの土台として扱い、画面体験の主役は単語ごとに変わる文字エフェクトに寄せます。

## コンセプト案

採用案: **モジリアクト**

- 意味: 文字がリアクトするタイピング。
- 理由: 入力方式ではなく、見た目の変化をゲームの顔にできる。
- 初回コピー: `打つたび、ことばの意味が文字に出る。`

他の候補:

- **ワードリアクト**: 意味は伝わりやすいが、少し説明的。
- **モジエフェクト**: 直感的だが、やや普通。
- **ことばリアクター**: 面白いが、少し長い。
- **リアクタイプ**: 短いが、意味が少し伝わりにくい。

## 方針

- ジャンルは「日常ワード」ひとつに絞る。
- 入力方式対応は当たり前の基盤として整える。
- 画面の差別化は、単語の雰囲気に合わせた文字の色、濃さ、揺れ、光り方で作る。
- 「薄い」「爆速」「秘密」「ごちゃごちゃ」「雨」「熱い」など、意味と見た目が一致する反応を増やす。
- 最初の画面では、プレイ前から複数の文字反応が見えるようにする。
- プレイ中の背景も、現在のお題のムードに合わせて色と流れが変わる。
- 速度は流線、雨は斜め線、炎や発光はパルス、混乱はジッターのように、ムードごとの専用演出を持たせる。
- お題は100語。薄い、爆速、雨、炎、氷、無音、混乱など、見た目のエフェクトに直結する言葉だけに絞る。
- 結果画面で称号を出し、SNS投稿用の結果テキストをコピーできる。
- OGP画像は `app/assets/ogp.svg` に置き、公開時のSNSカードで世界観を伝える。
- 外部API、外部CDN、ビルド工程は使わない。
